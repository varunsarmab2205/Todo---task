# Todo List App 📝

## Description
This project is a creative Todo List application built using HTML, CSS, and JavaScript. It allows users to add and manage daily tasks through a clean and modern interface.

## Features
- Add tasks instantly
- Simple and user-friendly design
- Glassmorphism UI effect
- Background image design
- Responsive layout
- Beginner-friendly code structure

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure

TodoApp/
│── index.html
│── script.js
│── README.md

## How to Run

1. Download or clone the repository
2. Open the project folder in VS Code
3. Install Live Server extension
4. Right-click on index.html
5. Click "Open with Live Server"
6. Type a task
7. Click Add button
